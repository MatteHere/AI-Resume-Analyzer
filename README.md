# 📄 AI Resume Analyzer

AI Resume Analyzer is a Python and Streamlit-based project that analyzes resumes, detects skills, calculates resume score, ATS score, and matches resumes with job descriptions.

## Features

- Upload PDF and DOCX resumes
- Extract candidate information
- Detect technical skills
- Calculate Resume Score
- Calculate ATS Score
- Match resume with job description
- Show matched and missing skills
- Visual analytics dashboard
- Final improvement recommendations
- Premium 3D user interface

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- PDFPlumber
- Python-docx
- Regular Expressions
- NLP techniques
